JavaScript API Demo pro GitHub Pages

Toto je demonstrace jednoduchého client-side API v JavaScriptu, které funguje na GitHub Pages.

Dostupné endpointy:

GET /

Vrací uvítací zprávu s časovým razítkem

GET /data

Vrací ukázková data s položkami

GET /status

Vrací stav API a seznam všech endpointů

GET /neexistuje

Test neexistujícího endpointu (vrací 404)

Jak používat:

// Použití API v JavaScriptu: const response = api.get('/'); console.log(response); // Nebo můžete vytvořit novou instanci: const myAPI = new SimpleAPI(); const data = myAPI.get('/data');
